# Ocean Voxels


![First Image](/media/GEBCO%203D.png)
![Second Image](/media/SubWithTraces.png)

Project OV is a simulation project designed to subdivide 3D space into voxels and map NetCDF Oceanographic data to indivisual volumetric cells for high resolution acoustic propagation modeling.

## Purpose
Due to My involvement at URI GSO and their involvement in acoustic modeling, I wanted to take my software engineering skills and what I have learned about acoustics and make a acoustic simulation that is similar to a 3D Bellhop model.

This project is very much in development and a working version, nor the results from the project will be publicly released until end of May 2026

## Features
- Subdivides 3D space into customizable voxel grids
- Maps NetCDF oceanographic data to each voxel
- Supports high-resolution modeling for acoustic propagation simulations

## Getting Started
1. Clone the repository
2. Install dependencies (see requirements.txt)
3. Run the main application
